# Next.js Latest Boilerplate

### Easy-Clean Next.js boilerplate

## How To Use
##### Clone Repo
```bash
git clone https://github.com/yusa-imit/next-latest-boilerplate.git
yarn
```
##### Start Dev Server
```bash
yarn dev
```
##### Start Build
```bash
yarn build
```
##### Start Production Server
```bash
yarn start
```
##### Start Linting
```bash
yarn lint
```
##### Start StoryBook
```bash
yarn storybook
```
##### Start Prettier
```bash
yarn prettier
```
##### Start Test
```bash
yarn test
```
